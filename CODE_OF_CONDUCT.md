# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — contributing code, opening issues, reviewing pull requests, or engaging in discussions — you agree to uphold this Code of Conduct in all project spaces and public representations of the project.

## Reporting

Concerns about conduct can be reported to **infosec@cyoda.com**. Reports will be reviewed promptly and handled confidentially. Project maintainers are obligated to respect the privacy and security of the reporter of any incident.

## Enforcement

The Contributor Covenant's enforcement guidelines apply. Maintainers will determine appropriate responses following the escalation path described in the canonical text linked above.
